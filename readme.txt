Программа для нахождения параметров прямоугольника.

Класс Rectangle: Определяет прямоугольник с методами вычисления периметра, площади и диагонали.

Методы:

calculatePerimeter(): Вычисляет периметр (2 * (ширина + высота)).

calculateArea(): Вычисляет площадь (ширина * высота).

calculateDiagonal(): Вычисляет длину диагонали с использованием теоремы Пифагора.

Ввод/вывод: Программа запрашивает у пользователя размеры прямоугольника и выводит результаты вычислений.
